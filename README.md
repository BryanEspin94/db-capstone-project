# Little Lemon Database Project

This is the capstone project for the Meta Database Engineer course. It includes:
- **ER Diagram**: The database structure.
- **SQL Scripts**: Queries for database setup and operations.
- **Reports**: Insights from the data analysis.

## How to Use
1. Clone this repository.
2. Run the SQL scripts in your preferred database environment.
3. Refer to the documentation for detailed usage.
